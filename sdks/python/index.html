Getting Started with the Python SDK

Rackspace's official SDK for Python is OpenstackSDK, an Apache licensed SDK for OpenStack clouds.

OpenstackSDK supports these Rackspace services directly through the use of the RackspaceSDKplugin.

  • Cloud Block Storage
  • Cloud DNS
  • Cloud Databases
  • Cloud Files
  • Cloud Images
  • Cloud Load Balancers
  • Cloud Monitoring
  • Cloud Servers

Using OpenstackSDK supports both Python 2.7.x and Python 3.5.x. We recommend installing pip to handle all of the dependency requirements for OpenstackSDK.

Installation

Generally, pyrax should be installed into a virtual environment using pip:

{% highlight sh %} $ pip install rackspacesdk {% endhighlight %}

This will install the RackspaceSDK plugin as well as the OpenstackSDK

Getting started with OpenstackSDK

{% highlight python %} from rackspace import connection conn = connection.Connection(username="user", api_key="b0f75295e3f844c18b678bdf7b6e0681", region="DFW") {% endhighlight %}

In addition to using the credentials within your script OpenstackSDK supports the use of the clouds.yaml file. You will want to create the clouds.yaml file in ~/.config/openstack/ and the file will need to look like this.

{% highlight yaml %} clouds: accountdfw: cloud: badger region_name: DFW accountord: cloud: badger region_name: ORD accountiad: cloud: badger region_name: IAD accountsyd: cloud: badger region_name: SYD accounthkg: cloud: badger region_name: HKG {% endhighlight %}

You will also want to have the clouds-public.yaml file setup. This will allow you to specify different Openstack Providers like Rackspace Public Cloud or a Rackspace Private Cloud environment. That file will look like the following.

{% highlight yaml %} public-clouds: badger: auth_type: rackspace_apikey auth: username: api-key: auth_url: https://identity.api.rackspacecloud.com/v2.0/ compute_api_version: 2 volume_api_version: 1 network_api_version: 2 {% endhighlight %}

With these 2 files configured you can then select different cloud endpoints or reagion using the following Python code.

{% highlight python %} import rackspace conn = connection.from_config(cloud_name='badgerdfw') {% endhighlight %}
